# Support

Optimisarr is maintained in spare time. There is no response-time commitment.

- For setup questions and ideas, open a GitHub issue and make clear that it is a
  question or proposal.
- For a reproducible fault, open an issue and include the image tag, your host
  OS, relevant Compose configuration with secrets removed, and the error from
  Queue or container logs.
- For a security issue, follow [SECURITY.md](SECURITY.md) instead of opening a
  public issue.

Before asking, check the [getting-started guide](docs/setup/getting-started.md),
[hardware guide](docs/setup/hardware-acceleration.md), and
[troubleshooting guide](docs/troubleshooting/diagnostics.md).
